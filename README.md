**MindEase+ (AI-Powered Mental Health App)**

**Overview**

MindEase+ is an AI-powered mental health application developed as a Final Year Project. The app is designed to provide accessible mental health support through intelligent features, personalized recommendations, and an intuitive user experience.

**Features**

AI Chatbot Support,
Mental Health Disorder Detection,
Habit Tracking,
Personalized Exercise Recommendations,
Firebase Authentication,
Cloud Firestore Integration

**Technologies Used**

Flutter,
Dart,
Firebase,
FastAPI,
Machine Learning

**Purpose**

The goal of MindEase+ is to make mental health support more accessible by combining mobile technology and artificial intelligence. The application helps users monitor their mental well-being, receive AI-assisted support, build healthy habits, and access personalized recommendations through a simple and user-friendly interface.

**Project Type**

Final Year Project (BS Computer Science)
